# 🚛 Générateur de Facture - Terratransport

## 📋 Description
Outil de génération de factures professionnelles pour l'entreprise **Terratransport** (importation de camions).

## ✨ Fonctionnalités
- Formulaire dynamique pour saisir les informations client et produits
- Ajout/Suppression de produits en temps réel
- Aperçu instantané de la facture
- Export PDF avec mise en page professionnelle
- Numéro de facture unique (FCT-AAAA-XXXX)
- Conversion automatique du montant en lettres
- Design élégant et responsive
